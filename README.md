A simple weather application built with React that shows current weather data and supports multiple languages (English & Arabic) using i18next.

Demo
(https://havadurumu9.netlify.app/)

Features

Fetches real-time weather data from OpenWeather API
Language switch (EN / AR)
RTL / LTR layout support
Date & time localization with Moment.js

Tech Stack

React (Vite)
Axios
i18next
Moment.js
Tailwind CSS

Setup

Clone the repository:
git clone https://github.com/dania8shaghouri/weatherApp
cd weather-app

Install dependencies:
npm install

Create a .env file in the root directory:
VITE_WEATHER_KEY=your_openweather_api_key

Run the project:
npm run dev

Notes
You need a free API key from: https://openweathermap.org
.env file is not included in the repository.
